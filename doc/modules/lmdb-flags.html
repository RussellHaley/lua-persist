Module lmdb-flags

Constant representation of the LMDB flags.

I would like to separate these out the same way as the lmdb documentation does.

Tables

RETURN_CODES LMDB return code defines as a constant table.
MDB ALL LMDB defines as a constant table.
TXN Transaction return codes.

Fields

REVERSEKEY use reverse string keys
DUPSORT use sorted duplicates.
INTEGERKEY numeric keys in native byte order: either unsigned int or size_t.
DUPFIXED with MDB_DUPSORT, sorted dup items have fixed size NOTE: Not applicable to lua-persist
INTEGERDUP with MDB_DUPSORT, dups are MDB_INTEGERKEY-style integers
REVERSEDUP with MDB_DUPSORT, use reverse string dups
CREATE create DB if not already existing
